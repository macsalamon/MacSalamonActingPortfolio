# Showreel URLs

Create a file called `showreel-urls.txt` in this folder with your 6 showreel embed URLs.

## Format:

```
General: https://www.filmmakers.eu/de/actors/matthias-salamon/video/106897?autoplay=false&background_color=none&iframe=v2&playlist=h
German: https://www.filmmakers.eu/de/actors/matthias-salamon/video/XXXXX?autoplay=false&background_color=none&iframe=v2&playlist=h
English: https://www.filmmakers.eu/de/actors/matthias-salamon/video/XXXXX?autoplay=false&background_color=none&iframe=v2&playlist=h
Polish: https://www.filmmakers.eu/de/actors/matthias-salamon/video/XXXXX?autoplay=false&background_color=none&iframe=v2&playlist=h
Commercial: https://www.filmmakers.eu/de/actors/matthias-salamon/video/XXXXX?autoplay=false&background_color=none&iframe=v2&playlist=h
Dramatic: https://www.filmmakers.eu/de/actors/matthias-salamon/video/XXXXX?autoplay=false&background_color=none&iframe=v2&playlist=h
```

## How to get the embed code from filmmakers.eu:

1. Go to your profile on filmmakers.eu or schauspielervideos.de
2. Click on each showreel
3. Look for the "Embed" or "Share" button
4. Copy the iframe URL (the part in `src="..."`)
5. Paste it here

I'll update the YAML files with these URLs automatically.
