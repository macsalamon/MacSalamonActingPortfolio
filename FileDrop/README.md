# FileDrop - Easy Content Upload

This folder is for easy file management. Drop your content here, and I'll help you integrate it into the site.

## Directory Structure:

### `headshots/`
- Drop all your headshot photos here
- Recommended format: JPG or PNG
- Naming: `01.jpg`, `02.jpg`, `03.jpg`, etc.
- One main hero headshot: `hero.jpg` (will be cropped to circle)
- Aspect ratio 3:4 works best for gallery photos

### `cv/`
- Drop your CV PDF here
- Will be used to extract content for YAML and for download button

### `showreel-info/`
- Drop a text file with your 6 showreel URLs from filmmakers.eu/schauspielervideos.de
- Format example:
  ```
  General: https://www.filmmakers.eu/de/actors/matthias-salamon/video/XXXXX?autoplay=false&background_color=none&iframe=v2&playlist=h
  German: https://www.filmmakers.eu/de/actors/matthias-salamon/video/XXXXX?autoplay=false&background_color=none&iframe=v2&playlist=h
  English: https://...
  Polish: https://...
  Commercial: https://...
  Dramatic: https://...
  ```

### `audio-samples/`
- Optional: Drop voice-over MP3/WAV files here if you want to include audio demos

## How to Use:

1. Clone this repo to your local machine
2. Drop files into the appropriate FileDrop folders
3. Commit and push changes
4. I'll help you integrate them into the site structure
